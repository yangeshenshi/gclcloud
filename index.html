项目概述

gcli2api 是一个开源项目，旨在将 Google Gemini CLI 转换为标准的 OpenAI 和 Gemini API 接口，提供企业级的多账号管理和负载均衡功能。

API 转换

将 Gemini CLI 转换为标准 API 接口，兼容 OpenAI API 格式

多账号管理

支持多个 Google 账号的凭证管理和自动轮换

云端部署

支持 Docker、Kubernetes 和各大云平台部署

核心特性

企业级的功能和性能，满足生产环境需求

智能负载均衡

自动在多个 Gemini 账号间分配请求，避免 API 限制

分布式存储

支持 Redis、MongoDB、Postgres 等多种存储后端

安全可靠

支持认证、限流、重试机制，确保服务稳定

API 兼容

完全兼容 OpenAI API 格式，易于集成

监控统计

提供详细的使用统计和性能监控

易于部署

支持 Docker、Kubernetes，一键部署

Docker Compose 部署

最简单快速的部署方式

部署步骤

1

安装 Docker 和 Docker Compose

确保系统已安装 Docker 20.10+ 和 Docker Compose 2.0+

2

下载配置文件

下载 docker-compose.yml 文件到本地目录

3

启动服务

运行 docker-compose up -d 命令启动服务

4

访问服务

通过 http://localhost:7861 访问控制面板

配置文件

version: '3.8'
services:
  gcli2api:
    image: ghcr.io/su-kaka/gcli2api:latest
    ports:
      - "7861:7861"
    environment:
      - PASSWORD=your_secure_password
      - REDIS_URI=redis://redis:6379
    volumes:
      - ./data/creds:/app/creds
  redis:
    image: redis:7-alpine
    volumes:
      - redis_data:/data

volumes:
  redis_data:
